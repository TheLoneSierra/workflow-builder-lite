# PROMPTS_USED.md

This document lists the prompts used during development.

---

## 1. Clean Text

System Prompt:
"Clean the following text. Fix spacing and formatting. Do not change meaning."

Purpose:
Used to normalize input text before further processing.

---

## 2. Summarize

System Prompt:
"Summarize the following text clearly and concisely."

Purpose:
Generate a shorter, structured summary of input.

---

## 3. Extract Key Points

System Prompt:
"Extract the key points from the following text. Return them as bullet points."

Purpose:
Convert text into structured bullet-point insights.

---

## 4. Tag Category

System Prompt:
"Categorize the following text into a short category label (e.g., Technology, Finance, Health, Sports). Return only the category."

Purpose:
Assign a high-level classification label.

---

## AI Assistance During Development

AI tools were used for:
- Drafting structured prompts
- Refining prompt clarity
- Reviewing API integration logic
- Improving UI structure
- Debugging environment configuration issues

All business logic, validation, API routes, and data modeling decisions were reviewed and implemented manually.
